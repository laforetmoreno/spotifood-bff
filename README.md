# Spotifood-bff

## Dependências

- Node.js 12+

## Desenvolvimento

Clonar esse projeto

1. `cd spotifood-bff`
2. `cp .env.sample .env` (E alterar as envs necessárias)
3. `npm install` or `yarn`
4. `npm start` or `yarn start`

## Rodando projeto no docker

1. `cd spotifood-bff`
2. `cp .env.sample .env` (E alterar as envs necessárias)
3. `docker-compose uo --build`

## Scripts

- `start` - Starts development server
