# Spotifood-bff

## Dependencies

- Node.js 12+

## Development

Clone this repository

1. `cd spotifood-bff`
2. `cp .env.sample .env` (And change the required envs)
3. `npm install` or `yarn`
4. `npm start` or `yarn start`

## Running with Docker

1. `cd spotifood-bff`
2. `cp .env.sample .env` (And change the required envs)
3. `docker-compose uo --build`

## Scripts

- `start` - Starts development server
